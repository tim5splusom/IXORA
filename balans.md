# Баланс

Это главная страница раздела финансов

На ней можно узнать:

* Общий баланс
* Баланс по договорам
* Остатки по лимитам
* Данные по общему обороту и график оборота
* Даты и размеры ближайших платежей
* Даты и размеры последних пополнений баланса

Также на этой странице можно запросить акт сверки и пополнить баланс

### Детали функциональности

{% column %}
В виджете "общий баланс" отображается текущий суммарный баланс по всем активным договорам.

Также кнопки пополнения счета картой или счетом на предоплату
{% endcolumn %}

{% column %}
Пополнить баланс можно картой или сформировать счет на предоплату.

Вы можете внести любую сумму вручную, оплатить только просроченные платежи, ближайшие платежи или закрыть всю задолженность.

{% column %}
В виджете "Лимиты" отражаются текущие остатки лимитов по активному договору, а также:

* Среднемесячный оборот за 3 месяца
* Сумма возвратов в работе
* Процент предоплаты для заказа
* Признак активного запрета или временного разрешения на отгрузку
