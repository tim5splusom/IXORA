# Самовывоз

{% hint style="warning" %}
Временно недоступно

Функциональность появится в одном из будущих обновлений

Следите за [poslednie-obnovleniya.md](../poslednie-obnovleniya.md "mention") и [dorozhnaya-karta.md](../dorozhnaya-karta.md "mention")
{% endhint %}

Если вы выберете самовывоз, будут показаны только товары, доступные для самовывоза.&#x20;

Активными наверху будут только позиции, которые можно доставить в выбранный пункт выдачи.

<figure><img src="../.gitbook/assets/image (73).png" alt=""><figcaption></figcaption></figure>

По умолчанию выбран пункт выдачи, куда можно доставить максимальное количество товаров из вашего списка. Вы можете изменить пункт, выбрав более удобный из выпадающего списка с адресами всех пунктов выдачи в вашем регионе. Рядом с каждым адресом указано, сколько товаров можно туда доставить.

<br>

<figure><img src="../.gitbook/assets/image (74).png" alt=""><figcaption></figcaption></figure>

Товары, доступные для самовывоза, но не из выбранного пункта, будут показаны ниже и неактивны.
