# Множественные корзины

### Общее

### Поведение

* Каждая корзина отображается отдельно, со своей таблицей товаров и условиями.
* Переключение между корзинами происходит через меню выбора корзин над панелью управления
* В каждой корзине доступен полный функционал: редактирование количества, добавление комментариев, фильтры и поиск.
* Общая стоимость считается отдельно для каждой корзины, заказ можно оформить только по товарам из одной корзины
